# ITI - Intake 46 - Bash Course - DBMS Project

## Project Description

This project is a **Database Management System (DBMS)** implemented using **Bash scripting** as part of the ITI Intake 46 Bash Course.
The system allows users to create databases, manage tables, insert/select/update/delete data, and execute basic SQL-like commands.

## Features

- Create / List / Connect / Remove Databases
 
- Create and manage tables

- Primary Key & Foreign Key support
 
- Insert, Select, Update, Delete records
 
- Simple SQL mode (SELECT queries)
 
- Backup database feature
 
- Fully implemented using **Bash scripting**
